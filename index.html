¿Qué es CSS?

Cascadeing Style Sheets

  • Posicionar elementos

Es una biblioteca pensada para simplificar la manipulación del DOM, eventos, animaciones y Ajax

Es software libre http://www.jquery.com

  • Interactividad y modificaciones del DOM
  • Soporte CSS1-3
  • Eventos
  • Efectos y animaciones
  • Ajax
  • Multi-Browser

CooffeeScript

¿Qué es CoffeeScript?

Lenguaje que "compila" a Javascript.

  • Sintaxis inspirada en Ruby, Python y Haskell
  • Mas legible
  • Mas expresivo
  • Crea mejor JavaScript (y mas pequeño)
  • Todo es una expresión
  • Permite el desarrollo de POO

¿Por que usarlo?

Básicamente, es más rápido, efectivo y Seguro 

class Animal
  constructor: (@name) ->

  move: (meters) ->
    alert @name + " moved #{meters}m."

class Snake extends Animal
  move: ->
    alert "Slithering..."
    super 5

class Horse extends Animal
  move: ->
    alert "Galloping..."
    super 45

sam = new Snake "Sammy the Python"
tom = new Horse "Tommy the Palomino"

Backbone.js

MVC en el lado del Cliente

Permite dar estructura a la aplicación web. Proveyendo Modelos (con eventos asociados), Colecciones, Vistas conectandose con la API Restfull existente... y JSON


http://www.backbonejs.org

¿Por qué usarlo?

Permite separar/ordenar nuestro código tal como lo hacemos (deberíamos) en el lado del servidor

¿Que gano con utilizarlo?

  • Modelos/Colecciones se comunican con el server
  • Vistas reaccionan a eventos y se comunican con Modelos/Colecciones
  • Router permite manipular secciones de nuesta SPA

Ember.js, AgularJS, Spine, CanjS

Para saber más http://www.todomvc.com

Person = Backbone.Model.extend({
	initialize: function(){
	alert("Welcome to this world");
	}
});

var People = Backbone.Collection.extend({
    model: Person
});

var person = new Person();
var person2 = new Person();

var personas = new People([person,person2]);

PersonasView = Backbone.View.extend({
    initialize: function(){
    	personas.fetch()
    	personas.bind('reset',function(){
    		this.render();
    	});
    },
    render: function(){
		var template = _.template( $("#personas_template").html(), {personas:personas} );
        this.el.html( template );
    }
});
